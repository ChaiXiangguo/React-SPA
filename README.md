## Getting Started
Checkout this repo, install dependencies, then start the process with the following:

```bash
> git clone https://github.com/allan2coder/React-SPA.git
> cd React-SPA
> yarn install or npm install
> yarn start or npm start
```

## Build the app

```bash
yarn run build
or
npm run build
```
